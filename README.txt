# ProgramacionDAW1
Estos son mis ejercicios de programación de DAW1.
Puede haber errores, y todos estan hechos pre-corrección.
Todos estan ajustados al temario dado en clase, sin ninguna cosa extra.
La clase Metodo es mi clase de utilidades personal, y aunque se puede utilizar perfectamente, es recomendable revisar todas sus utilidades primero.
Si estás leyendo esto significa que estás interesandote en la asignatura, y eso me alegra bastante :D
Si tienes cualquier duda siempre puedes preguntarme en clase o por tu linea de comunicacion favorita.
También me gustaría que usarais la parte de "Issues" de GitHub para comentarme problemas que veais.
